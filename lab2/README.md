## dependencies

下面是我的环境搭建过程。

1.  `curl -s "https://get.sdkman.io" | bash`
2.  `sdk install sbt`
3.  `sdk install java`
4.  `sdk install scala 2.12.13`
5.  verilator 推荐编译安装

尽管这个项目有一个 xmake, 但是 xmake 没啥用，只是服务于 lsp 的
